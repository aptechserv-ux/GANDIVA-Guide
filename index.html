Critical Warnings — Never Ignore
⚠
Never arrest on NATGRID data alone. NATGRID + CDR + Field Verification = Arrest. All three must happen together before any action is taken.
⚠
BOI Immigration is the ONLY authoritative source for 'has the accused fled India?' Never assume or conclude without running BOI Immigration Search.
⚠
Case ID is mandatory. Every NATGRID request requires AP-XXXXXXX. No Case ID = request rejected. Register FIR first.
⚠
In murder cases, do NOT wait for forensics — submit NATGRID request simultaneously. Digital trails deteriorate fast. Submit within 1 hour of FIR.
⚠
A 75–80% Gandiva match is a starting point, not confirmation. Field verification is always required to confirm or rule out the identity.
⚠
If owner sold the vehicle, get the buyer's contact and run that through NATGRID immediately. Do not stop at the first registered owner.
Priority Actions — Do These First
🔥
Submit NATGRID within 30 minutes of FIR in Missing Persons cases — the golden hour applies to intelligence, not just field response.
🔥
Mark PRIORITY for Crime Against Women and child cases. This enables faster processing by the IT Core Team.
🔥
Run Image Search and 94 BNSS notice in parallel for cybercrime cases. Do not wait for one to finish before starting the other.
🔥
In gang cases, run ALL recovered phones through NATGRID simultaneously to map the complete network. Never run sequentially.
🔥
For NDPS: run PAN Search not just for identity but to unlock bank accounts for asset seizure under Section 68-A NDPS Act.
🔥
For NI Act / cheque bounce: the mobile number printed on the cheque is usually the best ER Search starting point.
Proven Techniques — Use These
✔
A switched-off or even 10-year-old number still works for ER Search. NATGRID uses KYC records, not live SIM status.
✔
Never assume an old NBW is too old. 1992 warrants were successfully cracked using NATGRID in 2025. Old FIR photos return 90–98% confidence.
✔
All numbers switched off is NOT a dead end. IMEI tracking and metadata analysis still work. Request IMEI analysis even for inactive numbers.
✔
FASTag data maps exact escape route. Combine with CDR tower data for the same timeframe to cross-confirm location after the crime.
✔
Record Gandiva match percentage in case diary (e.g. "94.66% match from Telecom database"). This adds significant credibility in court.
✔
Family NPR details are the fastest backup when all known phone numbers of the accused are switched off or dead.
✔
Tower dump + NATGRID ER is the strongest combination when no eyewitness can identify the accused.
✔
For inter-state cases, write formally to local SO/SP with NATGRID data. A formal coordination letter gets faster cooperation.
✔
If BOI shows no recent exit — accused is still in India. This prevented a misdirected international manhunt in Cr.No.91/2024.
✔
For social media offenses: most people use real profile photos. Try Image Search first before issuing 94 BNSS notice to the platform.
✔
Crime vehicle number is the fastest entry point — even if abandoned, registration traces the owner chain back to the accused.
✔
Run CDR analysis and bank notice simultaneously for fraud cases — parallel actions save critical time in fast-moving financial investigations.
