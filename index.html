🔴 Critical Warnings — Never Ignore
⚠
Never arrest on NATGRID data alone. NATGRID + CDR + Field Verification = Arrest. All three must confirm before any action.
⚠
BOI Immigration is the ONLY authoritative source for 'has the accused fled India?' Never assume without running BOI.
💡
Note your Case ID (AP-XXXXXXX) before submitting any NATGRID request. Each submission must reference it or it will be rejected.
⚠
In murder cases, do NOT wait for forensics. Submit NATGRID within 1 hour of FIR — digital trails deteriorate fast.
⚠
75–80% Gandiva match is a starting lead, not a confirmed ID. Field verification is always required to confirm or rule out.
⚠
If owner sold the vehicle, get the buyer's contact and run through NATGRID immediately. Do not stop at the first owner.
🟡 Priority Actions — Do These First
🔥
Submit NATGRID within 30 minutes of FIR in Missing Persons cases — the golden hour applies to intelligence.
🔥
Mark PRIORITY for Crime Against Women and child cases — enables faster processing by IT Core Team.
🔥
Run Image Search and 94 BNSS notice in parallel for cybercrime — do not wait for one before starting the other.
🔥
In gang cases, run ALL recovered phones simultaneously through NATGRID. Never sequentially.
🔥
For NDPS: run PAN Search not just for identity but to unlock bank accounts for asset seizure (Sec 68-A).
🔥
NI Act / cheque bounce: mobile number on the cheque is usually the best ER Search starting point.
🟢 Proven Techniques — Always Use
✔
A switched-off or 10-year-old number still works for ER Search. NATGRID uses KYC records, not live SIM status.
✔
Never assume an old NBW is too old. 1992 warrants cracked in 2025. Old FIR photos return 90–98% confidence.
✔
All numbers switched off is NOT a dead end. IMEI tracking and metadata analysis still work. Always request IMEI.
✔
FASTag maps exact escape route. Combine with CDR tower data for the same timeframe for dual confirmation.
✔
Record Gandiva match % in case diary (e.g. "94.66% from Telecom database") — adds significant court credibility.
✔
Family NPR details are fastest backup when all known accused phone numbers are switched off.
✔
Tower dump + NATGRID ER = strongest combo when no eyewitness can identify the accused.
✔
BOI showing no recent exit = accused still in India. Prevented a misdirected international manhunt in YSR Kadapa.
